## Project Overview

Dashboard for **No Strings Attached** restaurant (BH 24 Salt Lake City Sector 2). A new monthly Excel export is received every 30 days and processed into a static HTML dashboard.

## Workflow

### Step 1 — Process new data
```bash
python3 process_data.py                     # auto-detects newest .xlsx in current dir
python3 process_data.py path/to/file.xlsx   # specify explicitly
```
Reads the Excel export, filters to `Status = 'Printed'` orders only, outputs `dashboard_data.json`.

### Step 2 — Serve the dashboard
```bash
python3 -m http.server 8080
```
Open `http://localhost:8080/dashboard.html`.

> The dashboard uses `fetch('dashboard_data.json')` so it must be served over HTTP, not opened as `file://`.

## File Structure

- `process_data.py` — Python data processor (requires `openpyxl`)
- `dashboard.html` — Single-file dashboard (Chart.js from CDN, no build step)
- `dashboard_data.json` — Generated by processor; committed or regenerated each month

## Excel Format

The Excel file has a 4-row header (restaurant name/address), then a header row at row 5 with columns:
`Order No., Client OrderID, Order Type, ..., Items, My Amount (₹), ..., Grand Total (₹), Payment Type, ..., Status, Created, Sequence Name`

Key columns used:
- **Status**: only `'Printed'` orders are included (4 `'Cancelled'` orders are filtered out)
- **Items**: comma-separated menu item names (no per-item quantities or prices)
- **Grand Total (₹)**: order-level revenue — cannot be split per item
- **Created**: `dd Mon YYYY HH:MM:SS` format

## Dashboard Charts

1. **Monthly Revenue** — bar (revenue) + line (order count) per month
2. **Revenue Distribution by Day of Week** — custom box plot (Q1/median/Q3/whiskers/mean) using a Chart.js plugin; transparent bars set the y-scale, the `boxplotDraw` plugin draws boxes via `chart._boxplotStats`
3. **Sales by Time of Day** — 4-hour windows; slots 3–5 (12:00–24:00) are highlighted gold, earlier slots are dimmed
4. **Top Selling Items** — sortable table of top 20 items; `Packing` and similar charge items are excluded via `EXCLUDE_ITEMS` in `process_data.py`

## Known Quirks

- `Chart.defaults.animation = false` is set globally — required for Playwright/screenshot tools to capture canvas content correctly
- The `boxplot-custom` dataset type is NOT a registered Chart.js type; the boxplot plugin detects `chart._boxplotStats` instead
- Items column has no quantity data — "quantity sold" = number of order appearances
- `Americano` and `Americano (Standard)` are separate items (menu rename mid-period)
- `Darjeeling Tea 1st Flush` and `NSA STG Tea Darjeeling Tea 1st Flush` are the same tea (renamed)
